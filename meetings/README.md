# Meetings

Processed meeting notes and raw transcripts.

Suggested structure:
```
meetings/
├── YYYY-MM-DD-meeting-title.md    # Processed notes
└── transcripts/                    # Raw transcripts
    ├── granola/                    # AI-generated summaries
    └── whisper/                    # Verbatim transcriptions
```

Use `/meeting` (if configured) to process meeting recordings automatically.
