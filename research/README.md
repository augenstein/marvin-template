# Research

Research inputs, outputs, and long-term reference material.

Suggested structure:
```
research/
├── inbox/       # Drop zone for raw captures (URLs, notes, ideas)
├── output/      # Processed research outputs
├── reference/   # Long-term reference docs
└── topics/      # Active research organized by topic
```

Use `/research` or `/deep-research` (if configured) to generate research outputs.
